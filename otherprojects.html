Foodorder App

A food order app is a simple e-commerce application that allows a user to select different products and add them to a cart based on their quantity. The cart page sums up the total amount of the items in the cart and allows the user to order the products after making the payment and specifying the best pickup location. When the payment is complete, the product is added to the order history on the account page

Features

Food order allows the user to filter products based on two categories; dessert and drinks and it also has a searchview where the user can search for a product based on price and product name.Food order has a signin and register page in which the user has to register or signin to make any order or add any item to the cart
